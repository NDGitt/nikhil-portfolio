Setup Guide: AI Thought Logger in n8n

(No Tech Skills Needed)

Complete n8n Workflow Overview

This is what you'll build: A complete automation that handles voice notes, AI processing, and data storage

This workflow automatically takes your thoughts from Telegram (typed text or voice notes), transcribes them, organizes them with AI, and logs them neatly into Google Sheets. At the end, your Telegram bot even confirms that your thought was saved.

What You'll Need Before You Start

Telegram

Telegram account + a personal bot (easy to create in Telegram)

n8n Account

n8n account (free cloud account or self-hosted)

Google Sheets

Google account with access to Google Sheets

OpenAI

OpenAI account (to handle transcription and AI summarization)

Need an API key? Watch this guide: How to get OpenAI API Key

1

Create a Telegram Bot

1

Open Telegram and search for BotFather.

2

Type /newbot and follow the instructions to name it.

3

BotFather will give you a Bot Token. Copy this — you'll use it inside n8n.

Tip: Keep your bot token secure and don't share it publicly!

Need more help? For a detailed step-by-step video guide on creating a Telegram bot, check out: https://www.youtube.com/watch?v=Li2xrdRYP5o

2

Prepare a Google Sheet

1

Open Google Sheets and create a new sheet.

2

Rename the first tab to Sheet1.

3

Add these column headers in Row 1:

Timestamp Title Summary Transcript Tags
4

Keep this sheet open — you'll paste its link into n8n later.

3

Connect Accounts in n8n

Inside n8n, you'll need to connect your accounts once:

1

Go to Credentials in the left menu.

2

Add the following credentials:

Telegram API (paste your Bot Token from BotFather)
Google Sheets OAuth2 (log in with your Google account and allow access)
OpenAI API (paste your API key from platform.openai.com)
4

Import the Workflow

1

Download the n8n workflow JSON file below.

2

In n8n, click Workflows → Import from File/Clipboard.

3

Upload the JSON file or paste its contents and save.

Get n8n Workflow

Choose how you'd like to get the complete n8n workflow JSON:

Download JSON File

Workflow JSON copied to clipboard! You can now paste it directly in n8n.

5

Connect the Pieces

The workflow already has the logic wired up. Here's what each piece does:

Telegram Trigger

Starts when you send a message or voice note to your bot

→

Voice Check

If voice note → gets file → OpenAI transcribes it
If text → just takes your message

→

AI Summarizer

OpenAI GPT creates title, summary, and 3–5 tags

→

Google Sheets

Adds everything with timestamp

→

Telegram Reply

Confirms success or reports errors

Note: You don't need to edit the logic — just make sure the credentials point to your accounts.
6

Update Sheet Links

In the nodes named Pre - Get row(s) in sheet, Post - Get row(s) in sheet1, and Log Thought into Sheet:

1

Replace the Google Sheet ID with your own

2

Copy it from the link in your sheet: the long string between /d/ and /edit

Example:

If your sheet URL is:

https://docs.google.com/spreadsheets/d/1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms/edit

Your Sheet ID is: 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgvE2upms

7

Activate and Test

1

Save your workflow.

2

Click Activate.

3

Open Telegram and send your bot a voice note or a text message.

4

Within a few seconds, check:

Your Google Sheet → A new row should appear
Your Telegram bot → It should reply with confirmation

That's It!

You now have a working AI Thought Logger:

Send Anytime

Send your ideas to your bot anytime

Auto-Process

They'll be transcribed, tagged, and organized automatically

Safely Stored

Everything is safely stored in Google Sheets for easy searching later
